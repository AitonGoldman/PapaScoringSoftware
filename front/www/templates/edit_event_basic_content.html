Queue Bump Amount
The number of spots a player will be "bumped" down a queue if they are not present when it is their turn to play.
Allowed Unused Tickets
The maximum number of unused tickets a player is allowed to have at one time. Note that this includes a ticket currently being played by a player.
Player Id Sequence Start
The starting player id. All player ids will follow this id. I.e. if this is set to 100, the first player that registers will be player 100, then player 101, etc.
